PromSec: Prompt Optimization for Secure Generation of Functional Source Code with LLMs

Problem: The challenge of generating secure and functional code with large language models (LLMs), which often produce insecure outputs.

Objective: Develop a method that optimizes prompts for secure code generation without sacrificing functionality.

Key Contributions: Introduced a gGAN + LLM loop for iterative prompt refinement, contrastive learning for improving output quality, and transferable prompt designs that generalize across models.

Methodology: Uses a feedback loop between a generative adversarial network (gGAN) and an LLM to iteratively enhance code security, with contrastive learning applied to distinguish secure and insecure outputs.

Results: Achieved a 50% reduction in security vulnerabilities, improved prompt efficiency by 20%, and provided cost-effective solutions for large-scale secure code generation.

Multi-Instance Adversarial Attack on GNN-Based Malicious Domain Detection

Problem: Current GNN-based detectors are vulnerable to adversarial domain manipulations, potentially allowing attackers to evade detection.

Objective: Create MintA, a multi-instance adversarial attack targeting the weaknesses in GNN-based malicious domain detection.

Key Contributions: Developed the MintA algorithm, which optimizes multiple domain instances to evade detection; introduced a black-box approach to simulate real-world attack scenarios.

Methodology: Utilizes a surrogate model to craft adversarial examples, optimizing perturbations in node features and edges. Implemented using DNS datasets for practical relevance.

Results: MintA achieved an 80% evasion success rate against state-of-the-art GNN models, with demonstrated robustness against various defense mechanisms.

SA-DS: A Dataset for Large Language Model-Driven AI Accelerator Design Generation

Problem: Existing datasets are insufficient for training LLMs in AI accelerator design, limiting the exploration of design possibilities.

Objective: Introduce the SA-DS dataset tailored for LLM-based design of deep neural network (DNN) accelerators.

Key Contributions: Provided a comprehensive dataset generated using the Gemmini framework, designed for LLM training in hardware design; introduced fine-tuning protocols using Verilator for simulation.

Methodology: Data collection from Gemmini, applying multi-shot learning techniques for LLM training, and fine-tuning with Verilator for accurate simulations.

Results: Achieved a 100% pass rate in design verification tests, with a 30% reduction in hardware design time and improved efficiency in LLM-assisted design processes.

Deepfake Detection with Vision-Language Model Explanations and GNN Integration

Problem: Deepfakes pose a growing threat to media authenticity, with existing detection methods often lacking robustness and generalizability.

Objective: Integrate vision-language models with GNNs to provide more reliable deepfake detection, utilizing both visual cues and textual explanations.

Key Contributions: Developed a method combining vision and language model outputs with GNN analysis, robust against adversarial attacks, offering detailed explanations for detected fakes.

Methodology: Applied a dual-layer analysis involving image features and AI-generated textual descriptions, coupled with graph techniques for relational analysis.

Results: Achieved a 15% increase in detection recall, enhanced resistance to adversarial deepfake attacks, and scalability across different datasets.

Semi-decentralized Inference in hetGNNs for Traffic Demand Prediction

Problem: Traditional GNN-based traffic prediction models face scalability issues, leading to high latency and computational overhead.

Objective: Develop a semi-decentralized inference method using heterogeneous GNNs (hetGNNs) to improve scalability and reduce overhead.

Key Contributions: Introduced hetGNN-LSTM for semi-decentralized traffic forecasting, using cloudlet-based architecture and adaptive clustering techniques.

Methodology: Utilized cloudlets to manage subgraphs and applied clustering algorithms for optimizing data flow and reducing communication overhead.

Results: Improved accuracy by 12%, reduced inference time by 25%, and demonstrated enhanced scalability in real-world traffic scenarios.
