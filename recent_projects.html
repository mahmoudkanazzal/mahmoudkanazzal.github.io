PromSec: Prompt Optimization for Secure Generation of Functional Source Code with LLMs

Problem: LLMs tend to generate functional but insecure code.

Objective: Optimize prompts to generate secure functional code with LLMs.

Key Contributions: gGAN + LLM loop, contrastive learning, transferable prompts.

Methodology: Iterative loop for optimizing code security using gGAN and LLM improvements.

Results: Enhanced security, reduced vulnerabilities, cost-effective for large-scale development.

Multi-Instance Adversarial Attack on GNN-Based Malicious Domain Detection

Problem: Need for a new adversarial attack to evade GNN-based malicious domain detection.

Objective: Develop MintA, an attack that optimizes multiple domain manipulations for evasion.

Key Contributions: MintA, black-box attack, optimizing node perturbations.

Methodology: Surrogate model, optimize node and edge perturbations, DNS-based implementation.

Results: Achieves 80% success in evading detection, robust against defenses.

SA-DS: A Dataset for Large Language Model-Driven AI Accelerator Design Generation

Problem: Lack of specialized datasets for AI-driven accelerator design.

Objective: Introduce SA-DS for LLM-based DNN hardware design.

Key Contributions: Diverse array designs using Gemmini, framework for hardware design optimization.

Methodology: Data from Gemmini, fine-tuning with Verilator, multi-shot learning.

Results: 100% pass rate, improved workflow, potential for LLM optimization.

Deepfake Detection with Vision-Language Model Explanations and GNN Integration

Problem: Increasing threat of deepfakes compromising media authenticity.

Objective: Combine visual and AI-generated textual analysis for robust detection.

Key Contributions: Visual-textual integration, robust to adversarial attacks, strong generalization.

Methodology: Dual-layer analysis with graph techniques, detailed textual explanations.

Results: Higher recall, stability under attack, scalable across diverse scenarios.

Semi-decentralized Inference in hetGNNs for Traffic Demand Prediction

Problem: Scalability issues in GNN-based traffic forecasting due to high overhead.

Objective: Create a scalable traffic forecasting method using semi-decentralized hetGNN-LSTM.

Key Contributions: HetGNN-LSTM, semi-decentralized architecture, adaptive assignment.

Methodology: Cloudlets manage subgraphs, clustering for reduced communication.

Results: Improved accuracy, reduced inference time, enhanced scalability.
